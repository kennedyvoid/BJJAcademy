# BJJ Academy — SwiftUI iOS

Versão 1.0 do app **BJJ Academy**.

## Recursos

- SwiftUI nativo
- Layout responsivo para iPhones
- Home com botão claro/escuro no topo
- Tema aplicado em todo o app
- Botão de tema aparece somente na tela inicial
- Posições permitidas por faixa
- Aba de técnicas proibidas/atenção por faixa
- Pontuação principal de competição
- Busca global
- Favoritos com AppStorage
- Cards com imagem + nome + pontuação
- Tela de detalhe com critérios
- Sem vídeos

## Como abrir

Abra o arquivo:

```bash
BJJAcademySwift/JiuJitsuGuide.xcodeproj
```

No Xcode:

1. Product > Clean Build Folder
2. Escolha um simulador de iPhone
3. Run
